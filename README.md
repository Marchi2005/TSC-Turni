# 📅 TSC Turni - Staff Management

Web App dedicata alla gestione operativa dei turni per **Tabacchi San Clemente Caffè**. 
Progettata per semplificare la pianificazione mensile tra i dipendenti, garantendo equità e flessibilità.

## ✨ Funzionalità Principali

* **📱 Mobile-First Design:** Interfaccia stile App, ottimizzata per l'uso da smartphone (aggiungibile alla Home).
* **🪄 Algoritmo 'Natural Flow':** Generazione automatica dei turni che evita pattern robotici, rispettando regole di equità e sequenze naturali.
* **☕ Gestione Festività:** Riconoscimento automatico di domeniche e festivi (inclusa Pasqua/Pasquetta) con logiche dedicate (es. Turno Unico o Entrambi).
* **🖨️ Stampa A4:** Layout CSS dedicato per generare un foglio presenze perfetto da appendere in negozio.
* **💾 No-Database:** I dati vengono salvati in locale sul dispositivo (LocalStorage) per la massima privacy e velocità, senza bisogno di server.

## 🚀 Come installare (Web App)

L'applicazione è accessibile via browser e installabile come PWA:

1. Visita il link: `https://[IL-TUO-USERNAME].github.io/tsc-turni/`
2. Su **iOS (Safari):** Tasto Condividi ⬆️ -> "Aggiungi a schermata Home".
3. Su **Android (Chrome):** Menu ⋮ -> "Installa app" o "Aggiungi a schermata Home".

## 🛠️ Tecnologie

* HTML5 / CSS3 (Variabili CSS per temi dinamici)
* Vanilla JavaScript (ES6+)
* LocalStorage API
